# Contributing to Simple Calculator 🧮

Thank you for your interest in contributing! We appreciate every bit of help—even fixing a typo 🎉

---

## 📌 Why contribute?

- Support fellow learners with helpful tools  
- Improve the project’s features and usability  
- Sharpen your HTML, CSS, and JavaScript skills  
- Practice working with Git and GitHub

---

## 🚀 Getting Started

1. **Fork** this repository  
2. **Clone** your fork:
   ```
   git clone https://github.com/<your-username>/simple-calculator.git
   ```
